## 1. Prerequisite
```
Before running the project, you must start the mongodb database server
```

## 2. Compile

```
$mvn clean install
```

## 3. Running

```
cd backend
$mvn spring-boot:run
```

## 4. Using Technologies
```
1.Angular4 + SpringBoot
2.MongoDB
3.Swagger UI for REST API Documentation
4.Logging
```
